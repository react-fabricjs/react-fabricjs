# react-fabricjs

A React component to fabricjs

## Install
```bash
yarn add @react-fabricjs/fiber
```

## Useage

```jsx
import { Canvas } from '@react-fabricjs/fiber'

export function() {
    return (
        <Canvas>
            <rfCircle width={100} height={100} radius={50} fill="red"/>
        </Canvas>
    )
}
```
